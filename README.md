# Econovation Networking

에코노베이션 네트워킹 행사용 모바일 우선 랜덤 질문 뽑기 앱입니다.

## 로컬 실행

```bash
npm install
npm run dev
```

## 배포 빌드

```bash
npm run build
```

## 질문 파일

- 루트의 `questions.md`는 빌드 시 예비 데이터로 포함됩니다.
- 실제 화면에서는 `public/questions.md`를 `/questions.md`로 불러옵니다.
- 질문을 수정할 때는 `public/questions.md`를 바꾸면 됩니다.
- 마크다운 제목, 구분선, 장식용 소제목은 자동으로 제외하고 질문 목록만 읽습니다.

## 중복 질문 처리

- 앱을 켠 뒤 이미 뽑힌 질문은 사용한 질문 목록에 저장됩니다.
- 모든 질문을 한 번씩 뽑기 전까지는 사용한 질문을 후보에서 제외합니다.
- 질문을 전부 사용하면 새 라운드를 시작하되, 직전에 나온 질문은 다시 나오지 않도록 피합니다.
- 새로운 라운드 버튼을 누르면 기록이 초기화됩니다.
- 새로고침하면 브라우저 상태가 초기화되므로 기록도 초기화됩니다.
